# beyond-infinity
A System For Managing Production Incidents And Outages. Teams Should Be Able To Create Incidents, Assign Responders, Post Live Updates, Maintain Timelines, Generate Postmortems, And Show A Public Status Page. Al Can Be Used For Summaries And Probable Root-Cause Suggestions.
